# Maskr.io Plugin for Penpot

AI-powered image processing tools for [Penpot](https://penpot.app), the open-source design platform.

## Features

- **Remove Background** - AI-powered background removal using BiRefNet
- More tools coming soon: Upscale, Inpaint, Segment, Icon Generation

## Installation

1. Open Penpot and press `Ctrl + Alt + P` (or `Cmd + Alt + P` on Mac)
2. Paste the manifest URL:
   ```
   https://maskrdotio.github.io/plugin-penpot/manifest.json
   ```
3. Click "Install"

## Usage

1. Select an image in your Penpot design
2. Open the Maskr.io plugin
3. Sign in with Google (first time only)
4. Click "Remove Background"
5. The processed image appears next to the original

## Requirements

- A [maskr.io](https://maskr.io) account (free tier available)
- Credits for processing (new accounts get free credits)
